# angularjs-web-component-pos

## Description
Demonstrates a POS, point of sale,
transaction of discounting items.

## Tech stack
- angular 1.6
  - components
- jquery
- popper
- bootstrap

## Docker stack
- httpd:latest

## To run
`sudo ./install.sh -u`  
http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
